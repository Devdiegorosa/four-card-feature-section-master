# 📊 Four Card Feature Section

A responsive feature section layout built with HTML and CSS.

This project is based on the **Four Card Feature Section challenge** from Frontend Mentor, where the goal is to recreate a modern UI layout using responsive design techniques.

🔗 Live Demo  
https://devdiegorosa.github.io/four-card-feature-section-master/

---

## 👨‍💻 About the Project

This project displays a section with four feature cards highlighting different functionalities of a fictional product.

The main goal of this challenge was to practice creating **multi-column layouts** and improving responsive design skills using modern CSS techniques.

---

## 🚀 Features

- Responsive layout
- Feature cards with icons
- Clean and modern design
- Mobile and desktop layouts
- Organized UI structure

---

## 🛠 Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox

---

## 🎯 What I Learned

While building this project I practiced:

- Creating layouts with **CSS Grid**
- Structuring responsive components
- Organizing card-based UI sections
- Improving spacing and alignment in layouts
